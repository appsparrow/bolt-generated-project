Score Keeper App v1.12

Release Date - February 2024

Home Page Features

  • View all existing games in a list format
  • Create new games with the "New Game" button
  • Delete existing games with 4-digit master passcode protection
  • Access game disclaimers and information
  • View comprehensive rummy rules

Game Setup Features

  • Secure 4-digit passcode protection for each game
  • Customizable game settings:
    • Drop Score (Default: 20)
    • Middle Drop (Default: 40)
    • Full Count (Default: 80)
    • Game Score/Out Score (Default: 251)
    • Winner Score (Default: 0)
  • Player management with sorting capabilities
  • Acknowledgment system with backend tracking

Rounds Features

  • Secure round access with master passcode
  • Real-time score tracking and updates
  • Visual score tracking with color-coded bar charts:
    • Yellow indicator for no-drop zone proximity
    • Red indicator for out status
  • Player rejoin system with automatic score adjustment
  • Round history and status tracking

Security Features

  • 4-digit master passcode protection
  • Game deletion confirmation system
  • IP address and timestamp tracking
  • Non-resettable game pins for enhanced security

Analytics Features

  • Visual performance tracking
  • Score progression monitoring
  • Player status indicators
  • Round history tracking

Known Limitations

  • Only one winner allowed per game
  • Game name cannot be blank
  • Game PIN cannot be reset once set
